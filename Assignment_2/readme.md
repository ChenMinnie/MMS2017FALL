## The Way To Set Up And Run Program

#### 1.讀取檔案
#### 2.找尋檔案數字相對的頻率
#### 3.將數值代入 "x(t) = Asin(2πft)" 公式， Fs = 8000, A =100
#### 4.輸出 .WAV 檔
#### 5.抓取使用者選取變音頻率值 f 
#### 6.將輸出的 .WAV檔 乘上 cos(2πft)，再輸出 .WAV 檔
#### 7.讀取 .WAV 檔並播放或停止
